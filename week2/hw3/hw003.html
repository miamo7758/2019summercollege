dcontent<-function(dcardurl,id){
  myurl<- paste0(dcardurl,id,collapse="")
  resdata<- jsonlite::fromJSON(httr::content(GET(myurl),"text",encoding = "UTF-8"))
  toString(resdata$content)
}
dcomment<-function(dcardurl,id){
  myurl<- paste0(dcardurl,id,'/comments?limit=100',collapse="")
  json<-httr::content(GET(myurl), "text",encoding = "UTF-8")
  resdata<- jsonlite::fromJSON(json)
  toString(resdata$content)
}
dseg<- function(str,term) {
  str = gsub("[A-Za-z0-9]", "", str)
  seg = cutter[str]
  seg = seg[names(seg)==term]
  id = which(nchar(seg) > 1)
  result = seg[id]
}

結論


除了國內的調查，國外研究也發現幾個導致青少年吸菸的因素

家庭史(本次主要討論因素)：
當家中有多位吸菸者時，青少年抽菸的頻率提高。家庭中，父母影響尤鉅；父母對抽菸的態度、觀感，親子關係都影響著青少年。父母失和、離婚、被家庭暴力傷害、亂倫、不成熟的父母、家庭連結薄弱都會增加風險。

陪伴者：
朋友或家人抽菸頻率高的話，青少年很可能就會步上後塵。尤其剛轉入一個新的學校或團體時，容易取得香菸以及早年開始抽菸，都會增加危險性。

個人特質或精神病理問題：
每個青少年都有其不同的價值系統、態度、所處環境、及個人知能（對學習及對社會）。具有反社會的、對學校反叛的、家庭關係或學業表現不佳的，很可能開始抽菸。有些青少年可能是為了要度過憂鬱或害怕而抽菸。

心理社會問題：
青少年可能認為抽菸是種大人的表現，可能想模仿明星或運動員。女孩開始抽菸，也有可能是想要擺脫社會對女性的刻板印象。

除了家庭以外還是有多因素會影響青少年吸菸的比率，但找到因素後再爭對每種因素去做防範，絕對是能更有效提升結果的!
